# 🔮 Obsidian AI — Personal AI Agent

Obsidian AI is a fully local, privacy-first personal AI agent 
that runs entirely on your own PC. No subscriptions, no API keys, 
no data sent to the cloud — just pure AI power on your machine.

## ✨ Features

- 💬 **AI Chat** — Powered by Llama 3.2 via Ollama, runs 100% locally
- 📎 **File Upload** — Attach and analyze .txt, .pdf, .csv, .json files
- ✅ **Task Manager** — Create, complete and track tasks with priorities
- 🔍 **Personal Search** — Web search via DuckDuckGo with AI summaries
- 📅 **Calendar** — Schedule and manage events with smart reminders
- 📊 **Weekly Report** — AI-generated productivity report every week
- 🔔 **Notifications** — Desktop alerts for tasks, events and reminders
- 🎙️ **Voice Input** — Speak your queries directly into the chat

## 🛠️ Built With

- **Backend** — Python, FastAPI, Uvicorn
- **Frontend** — HTML, CSS, JavaScript (no frameworks)
- **AI Model** — Llama 3.2 via Ollama (free, local, private)
- **Search** — DuckDuckGo API (no key needed)

## 🔒 Privacy First

Everything runs on your own machine. Your data never 
leaves your PC. No API keys. No subscriptions. No tracking.

## 🚀 Getting Started

1. Install [Ollama](https://ollama.com) and run `ollama pull llama3.2`
2. Install dependencies: `pip install -r requirements.txt`
3. Start backend: `uvicorn main:app --reload --port 8000`
4. Open `frontend/index.html` in your browser
